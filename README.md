# DataLearn — Free Data Analytics Learning Platform

**Learn SQL, Python, SAS, R, Excel, and dbt the way analysts actually use them.**

🌐 **Live app → [datalearn.us.com](https://datalearn.us.com)**

No account required. No installation. Open it and start learning.

---

## What is DataLearn?

DataLearn is a free, browser-based learning platform built for anyone who wants to break into data analytics or level up on the tools that real analysts use every day.

Every lesson is built around **real workplace scenarios** — not toy datasets or abstract exercises. You'll see the kind of problems that come up in actual analyst jobs and learn exactly how to solve them.

### What's covered

| Tool | Topics |
|------|--------|
| **SQL** | SELECT, JOINs, aggregations, window functions, CTEs, subqueries |
| **Python** | pandas, data cleaning, analysis, visualization |
| **SAS** | PROC steps, data manipulation, reporting |
| **R** | tidyverse, ggplot2, statistical analysis |
| **Excel** | VLOOKUP, pivot tables, formulas, Power Query |
| **dbt** | Models, refs, tests, documentation, best practices |

### Features

- Step-by-step lessons with explanations written in plain language
- Built-in code editor with syntax highlighting
- Hands-on challenges after every lesson
- Covers Beginner through Advanced topics
- Works on any device, no login needed
